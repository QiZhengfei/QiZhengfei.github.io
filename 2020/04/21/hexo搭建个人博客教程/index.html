title: 搭建个人博客流程记录

下载nodejs

nodejs下载地址，根据自己的系统类型下载.pkg文件，解压，按照步骤安装完成。

为npm添加淘宝镜像源cnpm

npm install -g cnpm --registry=https://registry.npm.taobao.org

安装hexo框架

cnpm install -g hexo-cli

搭建博客

进入想要搭建的博客目录，进行hexo初始化。（注意：如果下载速度很慢的话，可以使用终端代理或者用自己手机开热点，屡试不爽）

hexo init

启动博客，在本地查看博客是否成功搭建，localhost:4000。

hexo s

博客部署到github

打开github，新建一个项目，Repository name设置成：your_account.github.io

安装hexo部署到github的插件，

cnpm install --save hexo-deployer-git

打开博客目录下的文件_config.yml，到文件末尾进行修改。

保存。

修改之后，三连操作，就可以把博客部署到github上。（注意：每次修改都要三连操作）

这样，博客就部署到github上了，可以通过your_account.github.io直接访问。
